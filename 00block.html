텍스트 h2, p는 블록레벨이구나

블록레벨요소는 혼자 한줄을 차지해서 밑으로 떨어지는 요소

인라인 레벨 요소는 우측에 계속 컨텐츠가 붙음 span 은 인라인 요소

  • 인라인 요소 : 이미지
  • 블럭 요소 : 텍스트 태그 h2, p, li,
  • display:block; 인라인 요소를 블럭화함
  • display:inline; 블럭요소를 인라인으로 바꿔서 우측으로 떨어지게 수정함
  • display:inline-block; 요소를 인라인 레벨로 배치하면서 내용에는 블록레벨 속성을 지정, padding, margin등 안에서 속성지정할 때 inline과 다르게 사용함

hover 하면 transform, translateX / scaleX / rotate, rotate3d / skew

  • transform-origin : x축,y축,z축 initial inherit 특정 지점을 기준으로 회전 또는 변형 transform-origin:left bottom;
  • perspective: 크기 ㅣ none; 원근감을 갖게함, 0보다 커야하며 값이 클수록 사용자로부터 멀어짐
  • perspective-origin:x축 ㅣ y축 ; 입체적으로 표현할 요소의 아랫부분 위치 지정, 원근감을 줄때 기준점
  • trasform-style : flat 하위 요소를 평면으로 처리함, preserve-3d; 하위 요소들에 3d 효과를 같이 적용함
  • backface-visibility : visible 은 뒷면을 보이게 할거야, hidden은 보이지 않게
  • trasform:scale(2) perspective(120px) rotateX(180deg) 크기를 2배 확대하면서 x축 기준으로 180도 회전

트랜지션

  • transition-property :all l background-color l width, height 입력된 값만 트랜지션 할거야
  • transition-duration :2s , 1s 진행시간 설정, 시간단위는 s/ms
  • transition-timing-function:linear l ease l ease-in l ease-out 등 시작과 중간, 끝에서의 속도 지정
  • transition-delay: 1s 트랜지션이 언제부터 시작될지 지연시간 지정

스마트폰 세로 min-width:320px 스마트폰 가로 min-width:480px 태블릿 세로 min-width:768px, 태블릿 가로, 데스크톱 min-width:1024px

외부 css 파일 연결은 link 또는 import
link rel="stylesheet" media="screen and (max-width:768px)" href="css/tablet.css" (링크는 head에 삽입해야함) 화면 넓이가 768이하일때 적용, css가 많을때는 링크로 사용
@import url("css/tablet.css") only screen and (min-width:321px) and (max-width768px); (임포트는 style에 삽입) 화면 넓이가 321이상 768이하일 때 태블릿 스타일 시트 파일을 적용

style태그 안에 @media를 넣어서 쓰기도 함. 화면 너비가 320 이하일 때 배경색을 주황색으로 바꿀때는
@media screen and(max-width:320px){ body {background-color:orange}} 스타일안에 넣고 사용
