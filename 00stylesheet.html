시멘틱태그, 역할에 맞게 사용해야하는 태그

  • header: form으로 search영역을 넣기도 함, 상단 또는 왼쪽에
  • nav : 같은 사이트 안의 문서나 다른 사이트의 문서로 연결하는 링크를 나타냄, 푸터에 있는 사이트 링크 모음부분에도 사용됨
  • 본문 section : 주제별로 컨텐츠를 묶을 때, 섹선 제목을 나타내는 h2 이런 태그와 함께 사용
  • 본문 article : 컨텐츠 내용, 이부분만 떼어내 독립적으로 배포하거나 재사용 하더라도 완전한 하나의 컨텐츠
  • aside ; 본문 이외의 내용, 사이드 광고배너 같은 것들, 방문기록, 내프로필을 알려주는 좌우 사이드의 부가적 요소
  • iframe : 외부문서 삽입 <iframe src="주소"></iframe>, 안에 들어있는 내용을 따로 만들어서 호출하는 형태, 이부분만 수정할 수 있는 장점이 있음
  • address : 푸터에 회사정보 넣을때 쓰는 시멘틱태그
  • div는 컨텐츠를 묶어 시각적 효과를 적용할때, 즉 컨텐츠에 css를 적용할 떄 사용, 시멘틱태그로 틀을 잡고 div로 컨텐츠를 만듦

CSS선택자

컴비네이션 선택자
  • section a {} 하위 선택자
  • section > p {} 자식 선택자
  • h1 + ul {} h1 인접형제선택자 다음에 오는 첫번째 ul에만 적용
  • h1 ~ ul {} h1 형제선택자 다음에 오는 전체 ul 모두에게 적용
  • a[href] {} 속성선택자 a 태그 중 href 속성이 있는 요소를 찾아내 배경색 지정
  • a[target="_blank"] {} 속성=값 선택자, 주어진 속성과 속성값이 일치하는 요소를 찾아 스타일 적용
  • [class ~= "button"] {} 속성~= 값 선택자, class 속성 값에 button class가 포함된 요소에 적용
  • a[title l= "us"] {} a의 title에 us라는 단어가 있으면 적용, us- 가능, use는 불가능, 하이픈까지만 적용됨
  • a[title ^= "eng"] {} a의 title 값이 title="english" eng로 시작될 때 적용
  • a[href $= "xls"] {} a의 href값이 xls로 끝날때 적용, 주로 확장자 hwp, jpg등에 사용됨
  • a[href *= "w3"] {} 값의 일부가 일치하는 속성을 가진 요소를 찾음, a태그의 href중에 w3텍스트가 포함된 곳에 적용

CSS 주요속성

position:fixed/ 브라우저 기준으로 좌표를 입력해야함 top:5px; right:5px; 로 적용

visibility:hidden; 요소의 위치를 유지함, display:none;은 요소위치까지 숨겨짐

z-index : 숫자가 클수록 레이어가 위로 올라옴, z-index그룹을 relative로 설정해줘야함
