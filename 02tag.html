텍스트 관련 태그

  1. 로컬에 개발 폴더 임의로 만듦
  2. vs code에서 open으로 1의 폴더를 열어(맨위에 open...)
  3. 폴더가 열리면 우측에 폴더, 파일 추가가 생김, .html 등 확장자를 넣어 제목을 붙여주면 그대로 파일이 생성됨
  4. html화면에서 마우스 우측 open with live server 하면 go live가 실행됨~
  5. 깃에 올리는 방법
    • git init
    • git add README.md
    • git commit -m "first commit"
    • git branch -M main
    • git remote add origin https://github.com/jihes109/htmlguide.git
    • git push -u origin main
